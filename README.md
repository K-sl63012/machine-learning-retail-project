# Machine Learning Retail Customer Analysis

## Project Objective

To compare different Machine Learning classification algorithms using a retail customer dataset.

## Dataset Features

* Quantity
* UnitPrice
* Country

## Algorithms Used

* Logistic Regression
* Decision Tree Classifier
* K-Nearest Neighbors (KNN)

## Preprocessing

* Missing value handling
* Label Encoding
* Train-Test Split

## Evaluation Metrics

* Accuracy Score
* Confusion Matrix

## Results

| Model               | Accuracy |
| ------------------- | -------- |
| Logistic Regression | 100%     |
| Decision Tree       | 100%     |
| KNN                 | 99.999%  |

## Conclusion

The project demonstrates the complete Machine Learning workflow including preprocessing, training, evaluation, visualization, and model comparison.
